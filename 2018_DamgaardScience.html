2018_DamgaardScience

Package version: 2.0.0 (2022-01-28), Poseidon version: 2.5.0

Description: This package was initially constructed based on the Allen Ancient DNA Resource (https://reich.hms.harvard.edu/allen-ancient-dna-resource-aadr-downloadable-genotypes-present-day-and-ancient-dna-data, version 37.2). The package contains individuals from the first horse herders Inner Asia and Anatolia.

Contributors: Ayshin Ghalichi (ghalichi@shh.mpg.de), Clemens Schmid (schmid@shh.mpg.de)

Literature: Barros Damgaard et al. (2018)
